# This is a user manual to put the puzzle back together.

## General

The keys have two signs on them. A **number** and a **arrow**. **Always** insert key the **correct** way **with the arrow pointing towards the dot on the box**.

The keyholes have a dot marking the place the arrow should point and a greek letter. match the numbers to the greek letters the right direction.

The sign "^" markes a fragile paice, **be carefull**.
